Как подключить
  • 1Установите WhatsApp на телефон (Android или iOS) и войдите в аккаунт с нужным номером.
  • 2Нажмите Отсканировать QR ниже — QR-код появится на экране.
  • 3В WhatsApp на телефоне: ⋮ → Связанные устройства → Привязать устройство → направьте камеру на QR.
  • 4Статус сменится на Подключён. Клиенты смогут писать на этот номер — сообщения придут в Битрикс24.
Сессия держится несколько недель. Если телефон долго без сети — потребуется переавторизация через этот же экран.
MAX
Загрузка...
Как подключить
  • 1Зарегистрируйтесь на green-api.com/max — это бесплатный шлюз для MAX-мессенджера.
  • 2Создайте новый MAX-инстанс в личном кабинете. Скопируйте ID инстанса и Токен.
  • 3Откройте Настройки выше → вставьте ID и токен → нажмите Сохранить.
  • 4Нажмите Показать QR в карточке MAX → отсканируйте в приложении MAX на телефоне.
  • 5Статус сменится на Подключён. Сообщения на ваш номер MAX будут приходить в Битрикс24.
Бесплатный тариф — до 3 активных чатов. Для полноценной работы: ~690 ₽/мес на green-api.com.
Telegram
Загрузка...
Как подключить
  • 1Один раз: получите API-ключи на my.telegram.org → войдите тем же номером → API development tools → Create application. Скопируйте App api_id и App api_hash.
  • 2Добавьте ключи в Настройки (TG API ID и TG API Hash) → Сохранить → перезапустите Telegram-адаптер на сервере.
  • 3Введите номер телефона Telegram-аккаунта → получите код в SMS или уведомлении Telegram → введите код.
  • 4Если включена двухфакторная аутентификация — введите пароль 2FA.
  • 5Статус сменится на Подключён. Сообщения на аккаунт будут приходить в Битрикс24.
В России Telegram заблокирован — для работы сервера нужен VLESS-прокси (задаётся в Настройках). my.telegram.org открывайте с российского IP напрямую, не через прокси.

Как подключить
  • 1В Битрикс24: Приложения → Разработчикам → Другое → Локальное приложение. Название любое, тип — «Серверное приложение».
  • 2Права доступа: imconnector, imopenlines, crm, im. Укажите URL обработчика: https://ваш-домен/bitrix/events.
  • 3Скопируйте Client ID и Client Secret → вставьте в Настройки выше → Сохранить.
  • 4Нажмите Авторизовать приложение ниже — откроется страница Битрикса. Нажмите «Разрешить».
  • 5В Битрикс24 создайте Открытую линию (Контакт-центр → Открытые линии) и укажите её ID в Настройках.
ID Открытой линии — число в URL страницы линии в Битриксе.
